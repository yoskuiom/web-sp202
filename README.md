# Web Boss SP-202 Sampler q(≧▽≦q)
Веб-эмуляция лоу-фай семплера **Boss Dr. Sample SP-202**, 1998 г.
* проект доступен по ссылке https://sp-202-online.netlify.app/

## Стек 
* **React** 
* **Tone.js**
* **Tailwind CSS** 
* **Vite** 

## Локальная установка и запуск
ᓚᘏᗢ

1. клонировать репозиторий

```bash
git clone https://github.com/yoskuiom/web-sp202.git
```
2. перейти в папку проекта

```bash
cd web-sp202
```
3. установка

```bash
npm install
```
4. запуск

```bash
npm run dev
```

## Инструкция
### Включение и базовое управление
* Включение питания *Power Switch*
  - Переведите круглый тумблер Power в положение ON, чтобы активировать аудиодвижок веб-приложения. Без включения питания семплер не будет работать.
* Громкость *Volume Knob*
  - Левая вращающаяся ручка регулирует громкость. Вы можете нажать и перетаскивать её мышкой вверх/вниз (или использовать колесо прокрутки)
### Память
* Семплер поддерживает 4 банка памяти (по 8 семплов в каждом), разделенных на две группы:
  - "Встроенная память" Bank A & B *A/B (BANK INT)*
    - В Банке A предварительно загружены математически сгенерированные Lo-Fi семплы (барабаны 1-4 пэд, Rhodes-аккорды 5-6 пэд, синтезаторный бас 7 пэд и виниловый шум 8 пэд). Банк B по умолчанию пуст и готов к записи.
  - "Внешняя карта памяти" Bank C & D C/D *(BANK CARD)*
    - Слот SmartMedia (внизу слева): Нажмите на пустой слот карты памяти на нижней панели, чтобы вставить виртуальную карту SmartMedia 4MB. Карта плавно выдвинется, а светодиод покажет её наличие.
    - Если карта не вставлена, на экране загорится предупреждение Crd, и банки C/D будут заблокированы.
### Работа с пэдами (1-8)
На панели расположены 8 больших динамических пэдов.
* Воспроизведение семпла
    - Просто кликните по пэду. Если пад активен и на нем есть звук, он загорится красным цветом. Светодиодный дисплей отобразит текущую информацию.
* Ограничение полифонии
  - Семплер поддерживает динамическую 4-голосную полифонию. При запуске пятого звука первый (самый старый) автоматически прервется.
* Загрузка своих звуков
  - Если на пэде нет звука (надпись Empty), кликните по нему. Откроется стандартное окно выбора файла на вашем компьютере. Семплер мгновенно импортирует ваш аудиофайл (WAV, MP3, OGG и др.).
### Настройка параметров звука
Под экраном находится уникальная матрица светодиодов с 5 кнопками настройки параметров для выбранного (сфокусированного) пэда.
* GRADE (Качество звука).
  - Циклически меняет алгоритм декодирования звука:
    - HI-FI — чистое, стандартное качество
    - STANDARD — теплое звучание
    - LO-FI 1 — пониженная частота дискретизации и 8-битное сжатие
    - LO-FI 2 — экстремальный 4-битный перегруз с глухим "телефонным" фильтром
* OUTPUT (Канальность)
  - Переключает режим воспроизведения между:
    - MONO
    - STEREO
* PLAY (Режим воспроизведения):
  - TRIGGER — семпл проигрывается до конца от одного короткого нажатия
  - GATE — звук звучит только до тех пор, пока вы удерживаете нажатие на пэд
* LOOP (зацикливание) 
  - Переключает режим повтора LOOP (бесконечный цикл) или ONE SHOT (одиночный проход)
* REVERSE (Реверс)
  - Включает обратное воспроизведение семпла (REVERSE) или обычное (NORMAL)
### Встроенные эффекты EFFECTS и ручка CONTROL
* Правая поворотная ручка CONTROL отвечает за изменение параметров активных эффектов в реальном времени
* Кнопки EFFECTS:
  - PITCH (Изменение тона) — меняет тональность семпла в диапазоне от -12 до +12 полутонов без изменения скорости
  - TIME (Тайм-стретч) — меняет скорость воспроизведения семпла (от 0.5x до 1.5x) без изменения высоты тона
  - DELAY (Эхо/Дилей) —  настраивает время задержки обратной связи
  - FILTER 1 (Низкочастотный фильтр)— регулирует частоту среза классического мягкого low-pass фильтра
  - FILTER 2 (Резонансный Peaking-фильтр)—  создает винтажный резкий частотный пик ("sweep" эффект) для создания акцентов на средних частотах
  - RING MOD (Кольцевой модулятор): экстремальный металлический эффект модуляции, частота несущей волны настраивается ручкой Control от 30Гц до 1500Гц
### Кнопки режимов
* Del (Delete / Очистка и Обрезка)
  - Как удалить семпл с пэда: нажмите кнопку Del один раз (она замигает), затем дважды быстро нажмите на пад, который хотите очистить. На экране загорится CLR — пэд снова пуст
* Rec (Sampling / Живая запись)
  - Вы можете записать любой звук прямо со своего микрофона на любой пэд.
  - Как записать: нажмите кнопку Rec (экран покажет rdy). Затем нажмите на нужный пустой пэд. Начнется запись с вашего микрофона (на экране побежит таймер rEC). Снова нажмите кнопку Rec или Cancel для завершения записи. Звук мгновенно сохранится на выбранный пэд
* Mark (Truncate / Обрезка семпла)
  - Позволяет вырезать ненужные части аудиофайла
  - Как обрезать: во время воспроизведения звука нажмите Mark один раз — ручкой Control настройте начальную точку старта (S.xx). Нажмите Mark второй раз — ручкой настройте точку окончания воспроизведения (E.xx). Нажмите кнопку Del, чтобы применить изменения и обрезать семпл
* Cancel (Stop/ESC)
  - Экстренная кнопка остановки. Мгновенно глушит все запущенные циклы, останавливает текущую запись или сбрасывает выбранный режим
* Remain (Проверка памяти):
  - При нажатии на секунду симулирует винтажную проверку оставшейся памяти на устройстве, показывая случайное оставшееся время записи на экране ㄟ( ▔, ▔ )ㄏ
### Доп фишки управления
* BPM Tap Tempo
  - Большая серая кнопка TAP позволяет задать темп проекта вручную. Нажимайте её ритмично несколько раз, и семплер автоматически высчитает средний темп BPM, отобразив его на экране. Также вы можете корректировать BPM кнопками ▲ и ▼.
* Кнопка HOLD ⏸
  - При включении режима Hold (загорится оранжевым) пэды начинают работать в триггерном режиме "защелки": нажатие на пэд запускает его непрерывное звучание (даже если он был в режиме Gate), повторное нажатие останавливает его. Это идеально для запуска фонового винилового шума или ударных лупов
* Режим SOURCE 🎤:
  - При включении этой кнопки (загорится зеленым) сигнал с вашего физического микрофона компьютера начнет обрабатываться аудио-процессором семплера в реальном времени. Вы можете говорить в микрофон и крутить эффекты (например, Ring Mod или Pitch), превращая устройство в вокальный процессор эффектов!
