# Web Boss SP-202 Sampler q(≧▽≦q)
Веб-эмуляция лоу-фай семплера **Boss Dr. Sample SP-202**, 1998 г. 

## Стек 
* **React** 
* **Tone.js**
* **Tailwind CSS** 
* **Vite** 

## Локальная установка и запуск
```bash
# клонировать репозиторий
git clone https://github.com/yoskuiom/web-sp202.git
```
```bash
# перейти в папку проекта
cd web-sp202
```
```bash
#установка
npm install
```
```bash
#запуск
npm run dev
```

## Инструкция по использованию
1. Включение и базовое управление
Включение питания (Power Switch):
В верхней центральной части панели находится круглый тумблер Power. Переведите его в положение ON (загорится зеленым цветом), чтобы активировать аудиодвижок веб-приложения. Без включения питания семплер не будет реагировать на нажатия.
Громкость *(Master Volume Knob)*:
Левая вращающаяся ручка регулирует общую громкость веб-приложения. Вы можете нажать и перетаскивать её мышкой вверх/вниз (или использовать колесо прокрутки), чтобы изменить значение.
2. Использование банков звуков и карты памяти SmartMedia
Семплер поддерживает 4 банка памяти (по 8 семплов в каждом), разделенных на две группы:
Встроенная память (Банки A и B):
Переключение: Кнопка A/B (BANK INT) переключает внутренние банки.
Предустановки: В Банке A предварительно загружены фирменные математически сгенерированные Lo-Fi семплы (барабаны,Rhodes-аккорды, синтезаторный бас и виниловый шум). Банк B по умолчанию пуст и готов к записи.
Карта расширения (Банки C и D):
Слот SmartMedia (внизу слева): Нажмите на пустой слот карты памяти на нижней панели, чтобы вставить виртуальную карту SmartMedia 4MB. Карта плавно выдвинется, а светодиод покажет её наличие.
Переключение: Кнопка C/D (BANK CARD) переключает банки на внешней карте. Если карта не вставлена, на экране загорится предупреждение Crd, и банки C/D будут заблокированы.
3. Работа с падами (1 — 8)
На панели расположены 8 больших динамических падов.
Воспроизведение семпла: Просто кликните по паду. Если пад активен и на нем есть звук, он загорится красным цветом. Светодиодный дисплей отобразит текущую информацию.
Ограничение полифонии: Семплер бережно воссоздает аппаратные ограничения оригинала и поддерживает динамическую 4-голосную полифонию. При запуске пятого звука первый (самый старый) автоматически прервется.
Загрузка своих звуков: Если на паде нет звука (надпись Empty), кликните по нему. Откроется стандартное окно выбора файла на вашем компьютере. Семплер мгновенно импортирует ваш аудиофайл (WAV, MP3, OGG и др.).
4. Настройка параметров звука (Светодиодная матрица)
Под экраном находится уникальная матрица светодиодов с 5 кнопками настройки параметров для выбранного (сфокусированного) пада:
GRADE (Качество звука):
Циклически меняет алгоритм декодирования звука:
HI-FI — чистое, стандартное качество.
STANDARD — теплое звучание.
LO-FI 1 — пониженная частота дискретизации и 8-битное сжатие.
LO-FI 2 — экстремальный 4-битный перегруз с глухим "телефонным" фильтром.
OUTPUT (Канальность): Переключает режим воспроизведения между MONO и STEREO.
PLAY (Режим воспроизведения):
TRIGGER — семпл проигрывается до конца от одного короткого нажатия.
GATE — звук звучит только до тех пор, пока вы удерживаете клавишу мыши нажатой на паде.
LOOP (Зацикливание): Переключает режим повтора LOOP (бесконечный цикл) или ONE SHOT (одиночный проход).
REVERSE (Реверс): Включает обратное воспроизведение семпла (REVERSE) или обычное (NORMAL).
5. Встроенные Lo-Fi эффекты и ручка CONTROL
Правая поворотная ручка CONTROL отвечает за изменение параметров активных эффектов в реальном времени.
Доступные эффекты:
PITCH (Изменение высоты тона): Меняет тональность семпла в диапазоне от -12 до +12 полутонов без изменения скорости.
TIME (Тайм-стретч): Меняет скорость воспроизведения семпла (от 0.5x до 1.5x) без изменения высоты тона.
DELAY (Эхо/Дилей): Настраивает время задержки обратной связи.
FILTER 1 (Низкочастотный фильтр): Регулирует частоту среза классического мягкого low-pass фильтра.
FILTER 2 (Резонансный Peaking-фильтр): Создает винтажный резкий частотный пик ("sweep" эффект) для создания акцентов на средних частотах.
RING MOD (Кольцевой модулятор): Экстремальный металлический эффект модуляции, частота несущей волны настраивается ручкой Control от 30Гц до 1500Гц.
6. Продвинутые функции и кнопки режимов
В среднем ряду расположены кнопки управления режимами:
Del (Delete / Очистка и Обрезка):
Как удалить семпл с пада: Нажмите кнопку Del один раз (она замигает), затем дважды быстро нажмите на пад, который хотите очистить. На экране загорится CLR — пад снова пуст.
Rec (Sampling / Живая запись):
Вы можете записать любой звук прямо со своего микрофона на любой пад.
Как записать: Нажмите кнопку Rec (экран покажет rdy). Затем нажмите на нужный пустой пад. Начнется запись с вашего микрофона (на экране побежит таймер rEC). Снова нажмите кнопку Rec или Cancel для завершения записи. Звук мгновенно сохранится на выбранный пад!
Mark (Truncate / Обрезка семпла):
Позволяет вырезать ненужные части аудиофайла.
Как обрезать: Во время воспроизведения звука нажмите Mark один раз — ручкой Control настройте начальную точку старта (S.xx). Нажмите Mark второй раз — ручкой настройте точку окончания воспроизведения (E.xx). Нажмите кнопку Del, чтобы применить изменения и обрезать семпл.
Cancel (Stop/ESC):
Экстренная кнопка остановки. Мгновенно глушит все запущенные циклы, останавливает текущую запись или сбрасывает выбранный режим.
Remain (Проверка памяти):
При нажатии на секунду симулирует винтажную проверку оставшейся памяти на устройстве, показывая случайное оставшееся время записи на экране.
7. Дополнительные фишки управления
BPM Tap Tempo:
Большая серая кнопка TAP позволяет задать темп проекта вручную. Нажимайте её ритмично несколько раз, и семплер автоматически высчитает средний темп BPM, отобразив его на экране. Также вы можете корректировать BPM кнопками ▲ и ▼.
Кнопка HOLD (⏸):
При включении режима Hold (загорится оранжевым) пады начинают работать в триггерном режиме "Защелки": нажатие на пад запускает его непрерывное звучание (даже если он был в режиме Gate), повторное нажатие останавливает его. Это идеально для запуска фонового винилового шума или ударных лупов.
Режим SOURCE (🎤):
При включении этой кнопки (загорится зеленым) сигнал с вашего физического микрофона компьютера начнет обрабатываться аудио-процессором семплера в реальном времени. Вы можете говорить в микрофон и крутить эффекты (например, Ring Mod или Pitch), превращая устройство в мощный вокальный процессор эффектов!
